# 🍗 Cute Chicken Steak App 💖

A super cute animated recipe app built with Streamlit ✨

## Features
- 💖 Floating hearts animation
- 🐔 Cute doodle vibes
- 🔊 Optional sizzling sound
- 🍳 Interactive cooking steps

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
````

## Deploy on Streamlit Cloud

1. Push to GitHub
2. Go to https://streamlit.io/cloud
3. Select repo
4. Deploy 🚀
